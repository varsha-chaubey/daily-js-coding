What is a Debounce Function?

Debouncing ensures that a function is not called again until a certain amount of time has passed without it being called again. ðŸ§  Use Case: Useful when dealing with events like:

function debounce(func, delay) { let timer; return function (...arg) { clearTimeout(timer); timer = setTimeout( () => { func.apply(this, arg); }, delay ) } }
